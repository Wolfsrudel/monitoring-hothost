Configure your agent

Make sure you have Docker engine installed and run this: 

docker run -d \
  -v /etc/passwd:/host/etc/passwd:ro \
  -v /etc/group:/host/etc/group:ro \
  -v /proc:/host/proc:ro \
  -v /sys:/host/sys:ro \
  -v /etc/os-release:/host/etc/os-release:ro \
  --restart unless-stopped \
  --cap-add SYS_PTRACE \
  --security-opt apparmor=unconfined \
  -v /etc/hostname:/host/etc/hostname:ro \
  devforth/hothost-agent

Refresh this page to see update.

If you have a Docker Compose stack you can just add this snippet: 

version: '3'
services:
  netdata:
    image: devforth/hothost-agent
    restart: unless-stopped
    cap_add:
      - SYS_PTRACE
    security_opt:
      - apparmor:unconfined
    volumes:
      - /etc/passwd:/host/etc/passwd:ro
      - /etc/group:/host/etc/group:ro
      - /proc:/host/proc:ro
      - /sys:/host/sys:ro
      - /etc/os-release:/host/etc/os-release:ro

Refresh this page to see update.

If you want to run agent locally without Docker you can add one-time job on start crontab -e and add next 

@reboot HOTHOST_SERVER_HOST=1\
  HOTHOST_MONITOR_INTERVAL=1\
  HOTHOST_AGENT_SECRET=1\
  curl https://raw.githubusercontent.com/devforth/hothost/main/hothost-agent/getinfo.sh | bash
